Savory & Sweet 

Savory & Sweet is a simple and clean recipe website that I built as part of my frontend learning journey. The idea behind this project was to create a food website where users can explore different recipes like breakfast, lunch, dinner, and desserts in an organized way.

What this project includes

-> Homepage with featured dishes

-> Recipe listing page

-> Detailed recipe view with:

-> Ingredients

-> Step-by-step instructions

-> Prep time, cook time & total time

-> Category tags (Breakfast, Lunch, Dinner, Dessert)

-> Clean and responsive design
